# REGRESSORIUM

REGRESSORIUM is the terminal recourse plane of the Verifrax stack.

## What this repository is

REGRESSORIUM owns recourse-of-record.

It defines the point at which recognized truth is converted into attributable burden, admissible claim posture, binding remedy path, escalation route, and lawful closure.

**REGRESSORIUM resolves what must now follow.**

## What this repository is not

REGRESSORIUM is not:

- authority-of-record
- execution-of-record
- verification-of-record
- proof publication
- first-instance truth determination
- generic legal miscellany
- marketing trust center
- support desk

REGRESSORIUM does not create truth.
REGRESSORIUM does not replace terminal recognition.
REGRESSORIUM does not act as a verifier of first instance.

## Stack position

```text
SYNTAGMARIUM = law
ORBISTIUM    = state

TACHYRIUM    = cognition
AUCTORISEAL  = authority

```

## Boundary rule

```text
verification -> terminal recognition -> terminal recourse
```

## Adjacent sovereign surfaces

- **SYNTAGMARIUM** defines the law under which recourse is lawful.
- **ORBISTIUM** stores accepted state and historical epoch memory.
- **CONSONORIUM** reconciles drift and projection.
- **TACHYRIUM** transforms ambiguity into bounded machine-legible candidates.
- **VERIFRAX** establishes admissible verified structure.
- **ANAGNORIUM** determines the terminal recognition that precedes recourse.
