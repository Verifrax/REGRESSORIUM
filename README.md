# REGRESSORIUM

REGRESSORIUM is the terminal recourse plane of the Verifrax stack.

It determines burden, remedy, escalation, and closure of bounded claims without becoming law, accepted epoch, execution, verification, or recognition.

## Status

Active public sovereign boundary with an explicit terminal recourse role inside the Verifrax stack.

## Boundary

This repository owns terminal recourse only.

It determines how a recognized bounded claim proceeds through burden assignment, remedy selection, escalation, and closure.

It does not define law.
It does not hold accepted epoch truth.
It does not execute.
It does not verify.
It does not recognize claims.
It does not replace adjacent sovereign boundaries.

## What it does

- defines terminal-recourse doctrine
- defines counterparty, burden, remedy, escalation, and closure rules
- publishes recourse-object structure
- publishes current and historical recourse indexes
- publishes the current recourse-object entry surface
- preserves recourse as a distinct sovereign function
