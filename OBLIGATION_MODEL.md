# REGRESSORIUM Obligation Model

Obligation in REGRESSORIUM exists only where recognized truth has already crossed terminal recognition.

Required fields for any obligation model:

- recognition reference
- burden-bearing surface
- obligation class
- remedy path
- escalation path
- closure rule

## Machine-readable boundary surfaces

- `contracts/boundary-policy.json`
- `schemas/boundary-policy.schema.json`
- `schemas/recourse-output.schema.json`
- `schemas/recourse-audit-record.schema.json`
